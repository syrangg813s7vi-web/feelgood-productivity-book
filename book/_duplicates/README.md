# _duplicates

此目录用于存放 Feel-Good Productivity 项目中“章节编号重复”的历史文件。

## 为什么会有 duplicates
早期生成/恢复过程中出现同一章节号对应多个不同主题文件（如 Chapter22/23/24）。
为了让主目录 `book/BOOK_FULL_Chapter*.md` 保持 **1..32 章节一一对应**、便于统计与发布，把多余文件暂时归档到这里，不删除。
